AI赋能的IDE，GitHub Copilot推出免费套餐

NGX2024-12-222025-10-26

现如今，AI正在改变我们的生活方式，万事万物都具有AI，在软件开发方面当然也存在一批优秀的AI工具，如果将这些工具与IDE集成将会是一个提升开发效率的最佳方案。大概一年前，GitHub推出了GitHub Copilot，这是一个专为程序员提供的AI工具，但那时如果想使用，门槛非常高。现在，GitHub Copilot已经推出了免费套餐，大大提高了开发效率。

GitHub Copilot

现在打开 GitHub 屏幕最上方就有GitHub Copilot可供开发者使用。目前可免费使用 GPT4o（Azure OpenAI）和Claude 3.5 Sonnet。这两款模型都是世界顶尖的。

除此之外还可以与VSCode、JetBrains、XCode、Neovim等IDE进行集成。这里与VSCode和IDEA为例

与VSCode集成

去官网下载最新版的VSCode，里面会预装GitHub Copilot。

直接登录后就能使用

与JetBrains集成

JetBrains家的软件特别多，这里以编写Java程序的IDE——Intellij IDEA展示

打开IDEA，点击文件菜单→设置→插件 输入GitHub Copilot 并点击安装，等待安装完成

登录后就可以愉快使用了

一些限制

免费用户每个账号在一个月内可以使用最多2000 次代码辅助完成以及50 次聊天请求

如果账号多的话可以实现永久免费。

Cursor AI

这是一个免费的内置AI的IDE，发布时间比GitHub Copilot时间早。拥有的模型也很多，目前限制比GitHub Copilot少点。而且还提供2周Pro版本试用，如果账号多的话可以反复试用

使用方法：按住 Ctrl+L 可以对话，按住 Ctrl+K 可以生成

支持的模型要比GitHub Copilot多

在线使用

这不是官方提供的，这是一个为开发者提供的免费云电脑，里面预装了Cursor AI

引用站外地址
Cursor AI 在线使用
Browser-based Development - Quick Start

进入后等待大约40秒钟

使用方法一摸一样。

这可能是最全的 Serverless 平台的使用说明了

云服务器&云电脑Web前端云服务器技术

这可能是最全的 Serverless 平台的使用说明了

NGX2024-09-072026-01-02

无服务器运算（英语：Serverless computing），又被称为功能即服务（Function-as-a-Service，缩写为 FaaS），是云计算的一种模型。以平台即服务（PaaS）为基础，无服务器运算提供一个微型的架构，终端客户不需要部署、配置或管理服务器服务，代码运行所需要的服务器服务皆由云端平台来提供。

国内服务

腾讯云

引用站外地址
Serverless 应用中心-腾讯云
腾讯云

虽然方便，但是仅支持按量计费环境——也就是说，一键部署的环境，当免费资源用尽后，将会产生费用。且按量计费环境无法切换为包年包月环境。免费额度数据库读操作数只有 500 次 / 天。并且需要实名。

百度云

引用站外地址
函数计算CFC_无服务器计算-百度智能云
百度智能云

是付费的，免费套餐没了。需要实名。
资源套餐包，包含三种计费资源类型：资源使用量、函数调用次数和外网出流量。购买资源包后，调用函数时，将优先使用免费额度，当免费额度用完后则使用资源包抵扣。若无有效资源包或资源包已用完，则转为按需计费结算。详细资源套餐包价格请参见CFC价格详情。
按需付费，在您创建函数时无需付费，之后根据您使用函数时实际产生的调用次数、函数运行占用资源的时间和公网流量计费，具体的计费价格请参见CFC价格详情。
计算公式：总费用=函数调用次数费用+函数运行占用资源时间费用+公网流量费用

Zeabur

引用站外地址
部署服务从未如此简单
Zeabur

Zeabur是国内少数无需实名，无需付费的Serverless平台，有免费额度。

概括

可以看到，国内的 Serverless 平台通常需要实名，且价格不低。但是这些付费的服务通常在国内访问最快速，运行最稳定。免费的服务可能不是很稳定，访问速度通常会慢一些。

国外服务

Vercel

引用站外地址
Vercel: Build and deploy the best web experiences with the Frontend Cloud
Vercel

免费额度非常多，每月100G，无需实名，Github即可登录。默认的 vercel.app 域名被DNS污染了，请绑定自定义域名。
但缺点也很明显，国内访问速度太慢了，即使更改DNS到 cname-china.vercel-dns.com 也很慢。

Netlify

引用站外地址
Scale & Ship Faster with a Composable Web Architecture | Netlify
Netlify

和Vercel一样提供每月100G宽带，比Vercel稍微好那么一点点，但经常无缘无故封号。 netlify.app 域名在不同运营商、不同地区受到DNS污染。

HuggingFace Space

引用站外地址
Hugging Face – The AI community building the future.
HuggingFace

这是一个Ai平台，但是它提供了Space空间。这就可以当成一个 Serverless 来使用。并且允许允许通过 Cloudflare Tunnels 自定义域名。国内访问速度不错。

概括

这些国外的服务商基本都不需要实名认证，并且免费套餐很大方，但正是这种成本低、免费的特点，吸引了一大批违规网站。所以部分IP已经被相关部门屏蔽。在国内访问效果并不太好。如果你的业务在中国，那还是建议选择腾讯云，百度云等国内Serverless平台。
记住：免费的才是最贵的，所有的免费行为背后都有隐性的代价，所有免费的东西都标上了你看不见的价格。

可部署应用

Notion-Next

这是一个基于Notion的个人博客，笔记只需要在Notion上写好就可以自动同步过来。非常简单、方便，但性能不怎么好。适合零基础用户。

引用站外地址
NotionNext中文 | Notion笔记，轻松建站，NotionNext官网
Notion-Next

Github开源地址：tangly1024/NotionNext
演示站点（主题：HEO）：Tangly的学习笔记
文档：NotionNext帮助手册

Hexo

这也是一个个人博客平台，性能高，占用资源低。纯静态部署。功能较全但有一定的学习成本。本站就是基于Hexo搭建的。直接搭建Hexo是非常简单的，难点就是主题的选择、安装与配置。

Github开源地址：hexojs/hexo
演示站点（主题：AnZhiyu）：安知鱼 - 生活明朗 万物可爱
官方文档：Hexo
AnZhiyu文档：安知鱼主题官方文档

Waline

Waline是一款简洁、安全的评论系统。具有轻量完全免费部署、登录支持、自由评论、强大的安全性等特点，成为一众博客站长的最佳选择之一。

文档：快速上手
Github开源地址：walinejs/waline
演示站点：Waline Example

ChatGPT-Web-Next

这是一个在线的ChatGPT网页版，配合上API即可使用。部署后可以在国内使用，支持第三方OpenAI格式的API。拥有更多功能，更快地响应体验。

在线体验：Chat-Next-Web
一键部署：Vercel 或 Zeabur
环境变量：

变量 是否必填 说明
BASE_URL ✖ 第三方API端点值
OPENAI_API_KEY ✔ API值
CODE ✖ 访问密码（用英文逗号分隔）

Libre Chat

这个外观就特别像 ChatGPT。

演示Demo：LibreChat

网站首页：www.librechat.ai
